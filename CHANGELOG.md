## [v0.3.0] — 2026-06-09

### CHINHUNG 디자인 참고 — 전면 비주얼 리디자인

#### 개요
포크 원본(`aebonlee/rest03` CHINHUNG 건설사 사이트)의 디자인 언어를 동물병원 테마에 적용.
흰 헤더·풀와이드 히어로·필버튼·이미지밴드·큰 타이포그래피의 클린 모던 스타일로 전환.

#### 변경 사항

**컬러 시스템**
- `--color-primary`: `#0d3b55` → `#0e3d5c` (CHINHUNG brand 계열)
- `--color-accent`: `#4caf7d` 유지 (민트그린)
- 배경: 다크 섹션 → 화이트/라이트 배경 우선

**헤더 (CHINHUNG 스타일)**
- 다크 투명 → 항상 흰색 헤더 (`position: fixed`, `background: #fff`)
- 스크롤 시 그림자만 추가 (`.white` 클래스)
- 로고/네비게이션 항상 프라이머리 컬러
- `header class="dark"` 제거

**히어로 (KV)**
- 사이드 카드 패널 제거 → 풀-와이드 히어로 (`.card-wrap { display:none }`)
- 텍스트 크기: `clamp(32px, 4vw, 56px)` → `clamp(40px, 5.5vw, 88px)` (CHINHUNG급 임팩트)
- KV 네비 버튼: 원형 테두리 스타일 (CHINHUNG과 동일)
- 슬라이드 인디케이터 도트 추가 (활성 = 넓은 흰 바)
- JS: 슬라이드 변경 시 인디케이터 도트 동기화

**서비스 섹션 (진료과목)**
- 배경: 다크 프라이머리 → 화이트
- 타이틀: 흰색 → 프라이머리 컬러, 크기 업
- 카드: `border-radius: 8px` → `16px` (더 둥글게)
- 네비 버튼: 테두리 원형 스타일

**진료 철학 (CHINHUNG SustainabilityBand)**
- 기존: flex 레이아웃 회색 배경 → 풀-와이드 이미지 밴드
- `position: relative; height: 540px` + `.sustainable-bg` 이미지 + overlay
- 텍스트 크기 대폭 증가, 화이트 컬러
- CTA 버튼: 흰 필 버튼 스타일

**공지사항 헤더 (CHINHUNG MoreToDiscover)**
- 상단 구분선: `border-bottom: 2px solid #d1d5db` 추가
- 타이틀: 대형 폰트, 프라이머리 컬러
- 호버: 프라이머리 컬러로 강조

**온라인 예약 CTA**
- 배경: 다크 → 라이트그레이 (`--color-gray1`)
- 섹션 타이틀: 흰색 → 프라이머리 컬러
- CTA 버튼: 필 스타일, 프라이머리 배경

**CSS 구조 정리**
- `.pill-btn` 컴포넌트 추가 (CHINHUNG btn-pill)
- `.gototop` 박스섀도우: 구식 골드 → 민트그린 계열
- 전반적인 폰트 크기/굵기 상향 조정

## [v0.2.0] — 2026-06-09

### 테마 전면 개편: 건설회사 → 반려동물 헬스케어 동물병원

#### 개요
기존 건설/토목 회사 사이트 템플릿을 반려동물 헬스케어(동물병원) 홈페이지로 완전 재구성.
동일한 기술 스택(Vite, Swiper, GSAP)을 유지하고 컨텐츠와 디자인 시스템만 전면 교체.

#### 변경 사항

**브랜딩**
- 사이트명: `COMPANY` → `PawCare 동물병원`
- 슬로건: "당신의 소중한 가족, 최선의 의료로 지켜드립니다"

**디자인 시스템 (CSS 변수)**
- `--color-primary`: `#1a1a2e` (다크 네이비) → `#0d3b55` (딥 틸블루)
- `--color-accent`: `#c8a96e` (골드) → `#4caf7d` (민트 그린)

**네비게이션 구조 개편**
| 기존 | 변경 |
|------|------|
| 회사소개 | 병원소개 |
| 사업소개 | 진료안내 |
| 지속가능경영 | 건강관리 |
| 투자정보 | 온라인예약 |
| 고객센터 | 고객센터 |
| 인재채용 | 의료진 |

**서브메뉴 변경**
- 병원소개: 원장 인사말, 진료 철학, 시설 안내, 연혁
- 진료안내: 내과진료, 외과수술, 치과진료, 피부·안과, 영상의학
- 건강관리: 정기건강검진, 예방접종, 중성화수술, 영양관리
- 온라인예약: 진료예약, 상담신청, 진료시간 안내
- 의료진: 의료진 안내

**메인 섹션 개편**
- KV 슬라이드: 동물병원 컨셉 문구로 교체
- KV 고정배너: "전자조달 시스템/브랜드 홈페이지" → "온라인 예약/진료시간 안내"
- 창립 섹션 → 병원 소개 문구 (개원 스토리)
- 사업소개 → 진료과목 (내과/외과/치과/피부안과/영상의학)
- 지속가능경영 → 진료 철학 (Gentle Care, Trusted Partner)
- 공지사항: 건강검진 이벤트, CT 장비 도입 등 동물병원 소식으로 교체
- 투자정보 → 온라인 예약 CTA
- 푸터: 회사 주소 → 본원/분원 정보, 자매병원 셀렉트

**SVG 플레이스홀더 이미지 전체 교체 (10개)**
- KV 01: 틸-블루 그라디언트 + 발바닥/심전도 패턴
- KV 02: 그린 그라디언트 + 의료 십자 + 도트 패턴
- 카드 01~04: 내과/외과/예방접종/건강검진 아이콘 SVG
- 비즈니스뷰 01~05: 5개 진료과목 테마 SVG (청진기/수술/치아/눈/스캔)

**CSS 데코 텍스트 변경**
- `.sustainable::before`: `'ECO'` → `'CARE'`
- `.relation::before`: `'IR'` → `'BOOK'`
